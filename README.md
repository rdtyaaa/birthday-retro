# Birthday Retro Website

Welcome to **Birthday Retro**, a web project designed to create a memorable and interactive birthday celebration experience with a retro game-inspired theme.

## Features

- **Dialog Storytelling**: Interactive dialogues between characters (Octavia and Rascal) with typewriter effects.
- **Dynamic Background**: Animated backgrounds including particles and shooting stars.
- **Music Integration**: Background music and special tracks triggered at specific story points.
- **Interactive Elements**: Mini-games, stargazing, and hidden messages for added engagement.

## Technologies Used

- **Frontend**:
  - HTML5
  - CSS3 (with animations)
  - JavaScript (Vanilla JS)
- **Libraries and Tools**:
  - [particles.js](https://github.com/VincentGarreau/particles.js) for particle effects.
  - Custom animations and transitions for interactivity.

## Installation and Setup

1. Clone this repository:
   ```bash
   git clone https://github.com/rdtyaaa/birthday-retro.git
   ```
2. Navigate to the project directory:
   ```bash
   cd birthday-retro
   ```
3. Open `index.html` in your browser to view the project.

## How to Use

1. Follow the dialogues between Octavia and Rascal.
2. Make your birthday wish when prompted.
3. Enjoy the animations and music as the story unfolds.
4. Explore additional features like stargazing and hidden messages.
